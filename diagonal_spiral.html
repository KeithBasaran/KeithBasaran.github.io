Animating a Loop Pattern

Similar to the spiral path animation, this example uses the same techniques to generate a path, however it begins at the top of the canvas rather than from the centre.

The "moveTo" function is used to create the same parametric equation, while the "lineTo" function tracks the position of the path on the x and y axis. During every iteration the circle's position on the x and y axis is increased to ensure that the path is animated across the canvas.
